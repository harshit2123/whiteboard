# Whiteboard Application

A dynamic whiteboard application built with **React**, **HTML5 Canvas**, and styled using **Tailwind CSS**. This project features essential drawing tools and an intuitive undo/redo system, providing a smooth and efficient user experience. Deployed on **Vercel** for seamless access.

---

## 🔗 Live Demo
Check out the live application here: [[Whiteboard Application](#)](https://onlinewhiteboardproject.vercel.app/)

---

## ✨ Features
- **Drawing Tools**: Supports various shapes and tools like:
  - Line
  - Rectangle
  - Ellipse
  - Arrow
  - Text
- **Undo/Redo System**: Easily correct mistakes with an intuitive history mechanism.
- **Erasing**: Remove unwanted elements with precision.
- **State Management**: Powered by React's Context API for efficient state handling.
- **Responsive Design**: Styled with Tailwind CSS for a clean and adaptable interface.

---

## 🛠️ Technologies Used
- **Frontend**: React, HTML5 Canvas, Tailwind CSS
- **State Management**: React's Context API
- **Deployment**: Vercel
